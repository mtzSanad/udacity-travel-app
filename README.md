# Traveler Book

## Introduction

This app is a part of Udacity front end development nano degree, this is the last project in the nano degree and it aim to consolidae all the knowledge and principles learned during the whole nano degree, it tests the ability to correctly configure web pack project, writing test cases for front and back end, calling external apis from back end node server, separating critical keys into a separate file and finally dynamic rendereing the html and css.

## Installation steps

1- Clone the project

Then you need to install the dependencies using

```
    npm i
```

To run backend server

```
    npm run start
```

To run webpack-dev server

```
    npm run build-dev
```

For production run

```
    npm run build-prod
```

## Testing

To run project test cases

```
    npm run test
```
